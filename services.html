planning

  • Understand target audience
  • Create initial roadmap project
  • Select target platforms & devices

pre-production

  • Finalize product vision
  • Create documentation
  • Establish art direction
  • Outline code architecture
  • Early prototyping
  • Estimate & plan milestones

production

  • Core logic development
  • Art creation, modeling
  • Create mechanics, physics
  • Audio and VFX created
  • Create first playable build

alpha

  • All core features work
  • Save/Load system implemented
  • Art, models and UI integrated
  • Performance optimization
  • QA testing on target devices
  • Playtesting, balance testing

beta

  • Final UI/UX & performance optimizations
  • Platform-related services implemented
  • Multiple region support implemented
  • Compliance with all platform requirements
  • Code freeze & final bug fixing done
  • Build sent to partner for review & feedback

submission

  • Game is fully optimized
  • All critical bugs are fixed
  • Analytics & event system integrated
  • Build submitted for platform review
  • Compliance-related fixes if needed

  • Bug fixing for any issues
  • Monitor analytics & user feedback
  • Polish the game, improve KPIs

lunch

  • Hotfix of bugs and issues
  • UA\Marketing campaigns
  • Community management
  • Plan live-ops roadmap
  • Constant user support
