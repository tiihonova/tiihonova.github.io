game design roadmap

brainstorming

  • Establish the game’s genre
  • Define main design direction
  • Identity core audience
  • Create high-level game identity
  • Highlight main player motivations
  • Create game vision document

market research

  • Identify genre’s top games
  • Research competitors
  • Analyze market niche
  • Asses idea’s viability
  • Establish competitors’ flaws
  • Find possible improvements

game concept

  • Adjust the idea based on research
  • Select setting and story direction
  • Establish core loop
  • Create game mechanics
  • Identify unique selling points
  • Prepare GDD draft or pitch

pre-production

  • Create Game Design Document
  • Establish art direction
  • Create UI\UX wireframes
  • Plan and estimate production
  • Prototype new mechanics
  • Asses possible design risks

production

  • Update documentation
  • Create Technical Design Docs
  • Support production team
  • Playtest early playable builds
  • Game balance and economy
  • Compose texts for the game

submission

  • Game is fully optimized
  • Size optimizations if needed
  • All critical bugs are fixed
  • Build submitted for platform review
  • Compliance-related fixes if needed

launch

  • Playtest and adjust balance
  • Assist with bug fixing
  • Help promote of the game
  • User feedback analysis
  • Analyze game KPIs
  • Polish and improve the game

post-production

  • Plan updates
  • Create new content
  • Add new features
  • Game balance patches
  • Seasonal events
  • Live-ops activities
